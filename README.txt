Информация о проекте

Программа на Java - Необходимо написать программу, для розыгрыша игрушек в магазине детских товаров.


Подробное описание задач – ниже.


Как сдавать проект

Выкладываем в отдельный общедоступный репозиторий Github
Программа должна запускаться и работать, ошибок при выполнении программы быть не должно.



Задание

Необходимо написать программу – розыгрыша игрушек в магазине детских товаров.
Стараемся применять ООП и работу с файлами.
Если какой-то пункт не изучали и не знаете, как сделать, то можете сделать своим способом. Например, у кого в курсе не было ООП, то применяем списки\массивы\словари

Желательный функционал программы:
В программе должен быть минимум один класс со следующими свойствами:
id игрушки,
текстовое название,
количество
частота выпадения игрушки (вес в % от 100)

Метод на добавление новых игрушек и возможность изменения их веса (частоты выпадения игрушки)

Возможность организовать розыгрыш игрушек.
Например, следующим образом:
С помощью метода выбора призовой игрушки – мы получаем эту призовую игрушку и записываем в список\массив.
Это список призовых игрушек, которые ожидают выдачи.
Еще у нас должен быть метод – получения призовой игрушки.
После его вызова – мы удаляем из списка\массива первую игрушку и сдвигаем массив. А эту игрушку записываем в текстовый файл.
Не забываем уменьшить количество игрушек

К написанию программы можно подойти более творчески и делать так, как Вы поняли задание. Немного менять и отходить от примера выше.

Если будут вопросы-предложения, то пишите в личные сообщения

1. Создать класс Toy, который будет представлять игрушку. Класс должен содержать следующие свойства:
id игрушки, текстовое название, количество, частота выпадения игрушки (вес в процентах от 100).
2. Создать класс Store, который будет представлять магазин детских товаров. Класс должен содержать список игрушек
и методы для добавления новых игрушек, изменения веса и организации розыгрыша.
3. Создать класс Prize, который будет представлять призовую игрушку. Класс должен иметь методы для выбора призовой игрушки
и получения призовой игрушки после розыгрыша.
4. Создать файл для хранения списка игрушек. Файл может быть реализован как массив, список или словарь.
5. Реализовать методы для добавления новой игрушки в магазин и изменения ее веса.
6. Реализовать метод для организации розыгрыша игрушек. Розыгрыш может быть реализован с помощью генератора случайных чисел.
7. Реализовать методы для выбора и получения призовой игрушки.
Призовую игрушку можно выбирать случайным образом из списка игрушек или из массива призовых игрушек.

Описание классов:

– Класс “Игрушка”:
– id игрушки
– текстовое название
– количество
– частота выпадения игрушки (в процентах от 100).

– Класс “Розыгрыш”:
– список призовых игрушек
– метод выбора призовой игрушки
– метод получения призовой игрушки

Методы:

– Метод добавления новой игрушки
– Метод изменения частоты выпадения игрушки
– Методы организации розыгрыша
